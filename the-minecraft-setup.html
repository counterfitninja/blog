This post is to try and document all the different way I play and host minecraft servers so that i dont forget what i have done as time moves on. I play mainly Minecraft Bedrock edition but have recently switched between Java and Bedrock. This is primarily do to the Geyser plugins that allows Bedrock clients to connect to Java servers. 

Each of the links below will go into what kit i have, how i use each of them, and what config/ setup i have on each. 

What do i currently have and what do i play on:

Xbox Series X   ( Minecraft Bedrock) 

Windows 11 - Dell G15 Gaming Laptop 

Pixel 6Pro - Bedrock and occasional Proxy 

Dell Inspirion - Bedrock Server

Raspberry Pi - Phantom and Java/ Bedrock server ( small scale)
